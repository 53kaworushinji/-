# 羊城约咖
面向广州大学的泛熟人付费社交产品，核心功能为高效、安全地连接有经验分享需求的学长学姐与寻求指导、拓展人脉的学弟学妹，并通过线上预约、线下“Coffee Chat”的形式完成价值交换。
